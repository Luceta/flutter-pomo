## **Feature**

### Admin 페이지
- 유저가 10,15,20,25분 타이머를 설정할 수 있습니다.
- 4라운드가 끝나면 1골이 됩니다.
- 매 뽀모도로 타이머가 끝나면 휴식시간이 셋팅됩니다.
- 리셋을 누르면 진행되는 타이머가 전부 리셋되어집니다.
